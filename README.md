#### [48Live](https://github.com/Jarvay/48Live)的electron版
(web版大概IP被关小黑屋了？)

#### 安装步骤
``` bash
# 安装依赖
npm install

# 调试
npm run dev

# 构建
npm run build
```

#### 用到的一些开源项目
* [video.js](https://github.com/videojs/video.js)
* [flv.js](https://github.com/Bilibili/flv.js)
* [iview](https://github.com/iview/iview)
